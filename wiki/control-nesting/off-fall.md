# Off-fall and group settings

## Symptoms
- Control Nesting only allows one material size
- Freezing during nesting
- Inconsistent off-fall behavior

## Notes
- Change group settings to use off-fall only when available.
- Add multiple sheets so larger sheets can be selected.
- If needed, update `offfall.mdb` with additional sheet entries.
- A minimum off-fall change from `.5"` to `1"` resolved a freezing issue in one case.
