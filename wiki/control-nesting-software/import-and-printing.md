# Import and printing

## Import feature
Use the import feature to bring Control Nesting operating files into their designated locations when receiving export files from a customer.

### Ways to import
1. Hold the THM keys when starting the application.
2. Add `DontCheckTooling=true` under `[Settings]` in `machine.ini`.
3. Open Control Nesting and press `Alt + I`.

## Job level interface printing
- When using the Job Level Interface, labels and nest layout sheets can be printed similarly to eCabinets `.twd` jobs.
- Labels are created for each part and flip operation.
- Edgebanding data is not available in that workflow.
- If posting CNC code directly from another CAD/CAM system, contact that software vendor for barcode label options.

## Crystal Reports
- The labels report file can be edited in the Control Nesting and eCabinets report folders.
- The referenced file is `5366 Avery Labels.rpt`.
