# Pocket tool joinery

## Overview
Pocket screw joinery was added in Control Nesting version 5.78.

## Joinery behavior
- A 3/8 inch flat router bit creates the main angled pocket hole.
- The tool ramps down, drops to create screw head clearance, and ramps back out.
- An optional 1/8 inch to 5 mm flat tool can plunge a ledge hole for screw tip guidance.
- A cleanup pass may be used to reduce cracking or splitting in melamine or pre-finished surfaces.
- The method does not use a pilot hole through the part, so a self-drilling screw is recommended.

## Tool naming
- `pocketscrew` = 3/8 inch pocket tool
- `pocketscrewdrill` = position shelf drill only
- `pocketscrewdrills` = shelf drill plus cleanup operations

## Notes
- The screw tip sits in the ledge and the head lays on the ramped surface.
- Optional shallow spot holes can be used on the mating part to help guide the screw and reduce chipping.
