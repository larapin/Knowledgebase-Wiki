# System and version info

## Recommended desktop system
Recommended system for THM Desktop version:
- Operating systems: Windows XP, Windows Vista, Windows 7
- Hardware: about 200 MB of hard drive space

## THM software testing on Desktop version
To enable testing on Desktop version of THM software, place this in `C:\System\THM.ini`:

```ini
[Settings]
THM_Version_Override = 7.4.0.0
```

## Quick Cut option
- THM version 7.4 or newer must be installed on the controller
- eCabinet Systems version 6.0+ must also be installed
- Additional libraries may need to be loaded separately

## Manuals and part numbers
- QCore Manuals: MB00977
- GEN2 manuals include M40, M41, M42, M45, M53, M67, M70, M80, M83, M90, and Gen2 SuperControl manuals
- Keep the exact manual file in mind because some part numbers map to multiple file names

## Maintenance 3D Part Review
- Available on M45 machines in GEN II control versions 6.1.4.0 and later
- Access by pressing F10, F6, F12

## Daylight Savings Time note
Windows XP and 2000 Thermwood systems were affected by the 2007 daylight savings time change.
A Microsoft patch was required for Windows XP SP2 systems.
