# DNC operation

## Overview
Thermwood controls stopped supporting DNC networking in THM 5.05 / Windows 2000 controls.

## Recommendation
If a customer asks about DNC:
- Use the latest THM control as a stand-alone PC for machining operations.
- Use standard office networking to move files to and from the controller.

## Notes
- DNC was mainly used on older CNC machines with very little hard drive space.
- DNC drip-feeds coordinates to the machine as the part runs.
- Late-model controls usually have enough storage to run part programs directly from the controller.
- Running directly from the CNC control generally performs better than DNC drip-feed.
