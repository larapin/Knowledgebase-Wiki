# 5-axis setup and fixturing

## Typical remote programming inputs
For remote 5-axis programming, collect:
- Tool number intended for the program
- Tooling information, including overall length, flute length, and tool diameter
- Standard pivot distance
- A picture showing how the part sits on the fixture while trimming
- A picture of the part as it comes out of forming, so flashing and trim features can be understood

## Break-away head
On newer Thermwood 5-axis machines:
- Two 1/4-20 bolts can be inserted at the break-away point
- Bolts may be up to 2 inches long
- 1-3/4 inch bolts should also work
- The head must be rotated to access both bolts

## Onsite preparation
Before a tech arrives for onsite assistance:
- Have fixture stands built and ready for mounting
- Have the actual fixturing mold poured and mounted
- Have the D-seal groove ready to cut
- Have the vacuum supply line ready for quick attachment
- Have thermoformed parts available so the tech does not need to wait for parts to be formed

## General fixturing notes
- D-seal is useful for creating a vacuum seal for part hold-down
- Table vacuum should be plumbed to the fixture
- Quick disconnects are recommended
- Extruded aluminum tables use T-slots for fixturing
- T-nuts are commonly used for fastening
- Rayite 100 machinable media is useful for one-offs or demos, but not production

## Part fixturing methods
- Omer gun and Raptor polymer nails
- Toggle clamps
- Conventional vacuum with MDF spoil board and sealed pockets
- Universal vacuum with LDF/Trupan spoil board and high-flow vacuum

## Vacuum pumps and applications
- 36 HP: universal only
- 18 HP: universal only
- 15 HP: conventional, not recommended for universal but will work
- 5 HP: conventional only

## 5-axis machining guidance
- Fixture material as high off the table as possible
- Keep the Z axis near mid travel when possible
- Secure the material tightly to reduce chatter
- Use arcs and straight lines where possible
- Use tangency and acceleration factors appropriately
- Compression tools work well for plastics
- Low helix or 3 flute tools can also work well depending on material
